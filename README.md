# WordForge

> A Personal Vocabulary Operating System

WordForge is a lifelong vocabulary learning platform designed for researchers, language learners, and knowledge workers.

Rather than being a simple flashcard application, WordForge aims to become a complete vocabulary operating system that supports acquisition, review, knowledge organization, and AI-assisted learning.
